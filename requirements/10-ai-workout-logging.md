# AI-Assisted Workout Logging

## Use Cases

- Group classes (HIIT, CrossFit)
- No phone during workout
- Post-hoc reconstruction
- Voice or text descriptions

## Input Modes

- Voice dictation
- Typed free text
- Chat-style interaction

## Parsing Flow

1. User describes workout
2. AI extracts exercises, structure, timing, and intensity
3. Existing exercises are matched
4. Missing exercises are suggested for creation
5. User confirms and edits
6. Workout session is created and marked as AI-assisted

## Exercise Creation via AI

- AI proposes name, type, and muscle targets
- User confirms or edits
- Exercise saved as user-defined
